Individual Project Site Plan

Site Name

Xavi - Creative Web Designer & Developer

This name reflects my personal brand as a professional web designer and developer. It communicates creativity and technical skill to potential clients.

Site Purpose

The purpose of this website is to serve as an online portfolio where I can showcase past work and offer my design and development services to small businesses and startups. It will help me establish credibility and connect with future clients.

Scenarios

  • How can I view examples of previous projects?
  • How can I contact Xavi to start a new project?

Color Scheme

The website will use the following colors:

  • Primary color: #3498db – for headers, links, and buttons
  • Secondary color: #2c3e50 – used in the navigation bar and footer
  • Accent color: #e74c3c – for highlights and calls to action
  • Background color: #1a1a1a – general site background
  • Text color: #ffffff – primary font color on dark background

Typography

The website will use the Poppins font from Google Fonts for all text, including headers, body text, and navigation. It provides a modern, clean look suitable for professional portfolios.

Wireframe

The wireframe provides a basic structure for the layout of the home page in both mobile and desktop viewports. These are simple visual representations of how the content will be arranged on the page.
